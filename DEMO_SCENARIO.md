## 2. 데모 시나리오

### 시나리오 A: 복지 정보 검색 (RAG)

**상황**: 75세 김 할머니가 기초연금에 대해 알고 싶어함

1. **메인 화면** 진입
   - 따뜻한 베이지 톤의 화면
   - 큰 마이크 버튼 중앙에 위치
   - "안녕하세요, 김순자님" 인사말

2. **복지 정보 메뉴** 클릭
   - 퀵 메뉴에서 "복지 정보" 선택

3. **검색 실행**
   - 검색창에 "기초연금" 입력
   - 검색 버튼 클릭

4. **결과 확인**
   - 관련 복지 프로그램 카드 표시
   - 카드 클릭하여 상세 정보 확인
   - 혜택, 자격요건, 신청방법, 연락처 표시
   - "전화하기" 버튼으로 바로 연결 가능

### 시나리오 B: 음성 대화 (TTS)

**상황**: 어르신이 음성으로 날씨를 물어봄

1. **마이크 버튼** 클릭
   - 음성 채팅 모달 오픈
   - "말씀해 주세요" 안내

2. **음성 입력** (실제로는 텍스트 입력으로 대체)
   - "오늘 날씨 어때?"

3. **AI 응답**
   - 텍스트 응답 표시
   - TTS로 음성 재생
   - 느린 속도, 따뜻한 톤의 한국어 음성

4. **대화 계속**
   - 추가 질문 가능
   - 대화 기록 스크롤

### 시나리오 C: 정서 케어 대화

**상황**: 어르신이 외로움을 느끼고 이야기를 나누고 싶어함

1. **대화 시작**
   - "요즘 좀 외로워요"

2. **공감 응답**
   - "그런 마음이 드셨군요. 제가 곁에 있어요."
   - 따뜻하고 위로하는 톤

3. **추가 대화**
   - "이야기 나누고 싶으시면 편하게 말씀해 주세요"

## 3. 주요 포인트 강조

### 기술적 특징
1. **LangGraph 멀티 에이전트**
   - Supervisor가 의도 파악 후 전문 에이전트 라우팅
   - Welfare Agent: 복지 정보 RAG
   - Companion Agent: 정서 케어
   - Daily Agent: 일상 정보

2. **RAG 기반 복지 검색**
   - ChromaDB 벡터 스토어
   - 15개 복지 프로그램 데이터
   - 유사도 기반 검색

3. **음성 인터페이스**
   - STT: Upstage Whisper / OpenAI Whisper
   - TTS: Edge TTS (한국어, ko-KR-SunHiNeural)
   - 노인 친화적 음성 설정 (느린 속도, 차분한 톤)

### 노인 친화적 UI/UX
1. **큰 글씨**: 기본 18px 이상
2. **큰 버튼**: 터치 타겟 48px 이상
3. **높은 대비**: 명확한 색상 구분
4. **따뜻한 색상**: 오렌지 + 베이지 조합
5. **명확한 피드백**: 애니메이션으로 상태 표시

## 4. 예상 질문 & 답변

### Q: 왜 노인을 타겟으로 했나요?
A: 초고령 사회에서 디지털 소외 문제 해결. 음성 기반 인터페이스로 접근성 향상.

### Q: 실제 복지 데이터는 어디서 가져오나요?
A: 현재는 샘플 데이터, 추후 정부24/복지로 API 연동 예정.

### Q: 위기 상황(자해, 학대 등)은 어떻게 처리하나요?
A: 키워드 감지 후 전문 상담 연결 (129 복지콜센터, 112 등).

### Q: Upstage Solar Pro 2를 선택한 이유는?
A: 한국어 특화 LLM으로 어르신과의 자연스러운 대화에 최적화.
